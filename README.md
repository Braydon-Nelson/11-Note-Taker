## Getting Started

1. Clone the repo:
https://github.com/Braydon-Nelson/11-Note-Taker.git

2. Install the dependencies:

npm install

3. Run the app:

node server.js

Or, you can go to my live application:

https://uofu-11-note-taker.herokuapp.com/

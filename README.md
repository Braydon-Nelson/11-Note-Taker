## Getting Started

1. Clone the repo:

https://github.com/saltamay/UofT_BootCamp_NoteTaker.git

2. Install the dependencies:

npm install

3. Run the app:

node server.js

Or, you can go to my live application:

https://uoft-bootcamp-notetaker.herokuapp.com/
